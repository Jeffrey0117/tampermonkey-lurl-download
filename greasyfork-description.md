# LurlHub 助手 — lurl / myppt 全自動破解 + 下載 + 過期救回

> 連結過期了？密碼不對？LurlHub 全部搞定。

## 核心功能

### 1. 密碼自動破解

腳本會自動偵測頁面的上傳日期，換算成 MMDD 格式密碼，幫你自動填入、自動送出。

- lurl.cc — Cookie 注入，無需手動輸入
- myppt.cc — 表單自動填入 + 自動提交
- 零延遲，進頁面就破解

### 2. 一鍵下載影片 / 圖片

頁面上方自動插入下載按鈕，點一下就存到本地。

- 影片：直接抓取 MP4 原始檔，瀏覽器內下載
- 圖片：偵測所有預載圖片，支援「一鍵打包下載」（多圖頁面自動編號 \_1, \_2, \_3...）
- 支援 JPG、PNG、GIF、WebP
- 影片播放器自動解鎖（移除右鍵限制、啟用全螢幕控制）

### 3. 過期連結救回（LurlHub 獨家）

**這是我們最強的功能。** 別人的腳本只能破密碼，我們能救過期連結。

- 連結過期 → 自動查詢 LurlHub 備份庫（30,000+ 筆資源）
- 有備份 → 頁面上出現「一鍵救回」按鈕，點下去直接看
- 已救過的內容 → 下次進入自動載入，不扣點
- 密碼錯誤 → 不需要密碼，直接用備份觀看
- 影片載入失敗 → 自動偵測 + 自動切換備份源

免費用戶每月 3 次救回額度。需要更多？訂閱方案往下看。

### 4. Dcard 無縫整合

在 Dcard 西斯板 (dcard.tw/f/sex) 自動啟動：

- 年齡驗證自動通過
- 登入彈窗自動關閉
- 點擊 lurl / myppt 連結 → 自動帶入文章標題 + 來源網址
- SPA 換頁自動偵測重載

## 進階功能

### 離線佇列

網路斷線？沒關係。腳本會把擷取任務存入 IndexedDB，恢復連線後自動同步。

- 斷線 → 暫存 → 上線 → 自動補傳
- 最多 5 次重試，失敗自動歸檔
- 7 天自動清理過期任務
- 左下角狀態指示燈：即時顯示佇列狀態

### 大檔分塊上傳

影片太大？腳本自動分塊處理：

- 每塊 10MB，最多 4 塊同時傳
- 支援斷點續傳
- 不怕上傳中途斷線

### 智慧標題辨識

腳本會自動從多個來源提取正確標題：

`網址參數 → sessionStorage → 頁面標題 → og:title → meta title → URL slug`

不再出現一堆「untitled」的檔案。

### 版本自動檢查

- 自動偵測新版本，彈窗提示更新
- 重大安全更新時強制升級
- 24 小時延後提醒（非強制時）

我們非常重視你的隱私：

## 常見問題

**Q: 為什麼密碼破解失敗？**
A: 並非所有頁面都使用日期密碼。非日期密碼的頁面，腳本會自動切換到備份模式。

**Q: 救回次數用完了怎麼辦？**
A: 免費版每月 3 次。訂閱 BASIC 或 PREMIUM 可獲得更多額度。在救回彈窗中即可操作。

**Q: 腳本會影響瀏覽速度嗎？**
A: 不會。核心邏輯在頁面載入完成後執行，備份查詢使用索引查詢（毫秒級回應）。

**Q: 過期連結一定能救嗎？**
A: 要看 LurlHub 是否曾備份過該內容。目前備份庫有 30,000+ 筆資源，覆蓋率持續增加中。如果沒有備份，會顯示「此內容尚無備份」的提示。
